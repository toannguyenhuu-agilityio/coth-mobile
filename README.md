# COTH Mobile App

This is the mobile application for the COTH project, built using **Expo** and **React Native**.

## Branch Strategy

| Branch    | Purpose               |
| --------- | --------------------- |
| `master`  | Production-ready code |
| `staging` | QA / test builds      |
| `develop` | Active development    |

## Tech Stack

- Expo (React Native)
- TypeScript
- EAS Build (planned)
- Turborepo support (planned)

## Getting Started

```bash
# install deps
npm install

# run app
npm run start
```
